Pivot & Melt

Funciones para cambiar la estructura del DF

Pivot table

Transforma los valores de determinadas columnas o filas en los indices de un nuevo DF y la interseccin es el valor resultante.

Pivot Un parmetro intenresante de esta funcin es el fill_value: el cual nos permitir determinar con que contenido queremos visualizar los valores NaN, para este ejemplo uso el caracter " - "

Este es un ejemplo de cmo suelo usar melt() Digamos que tengo una tabla con los locales (filas) y las ventas anuales (cols). Y quiero como resultado - tener 3 cols con Local, Ao y Ventas.

Codigo

import pandas as pd
data = {
    'Local': ['Local A', 'Local B', 'Local C'],
    #Ventas por ao
    '2021': [100, 200, 300],
    '2022': [150, 250, 350],
    '2023': [200, 300, 400]
}
df = pd.DataFrame(data)

Luego convertimos a las 3 cols:

df.melt(id_vars=['Local'],
        var_name='Ao',
        value_vars=['2021','2022','2023'],
        value_name='Ventas')
